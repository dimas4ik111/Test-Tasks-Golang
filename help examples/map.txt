Что-то о Map (хэш - таблицах)

// инициализация
m := make(map[string]int)
// запись
m["habr"] = 42
// чтение
val := m["habr"]
// чтение с comma,ok
val, ok := m["habr"] // ok равен true, если ключ найден
// итерация
for k, v := range m { ... }
// удаление
delete(m, "habr")

Решение делать или нет map потокобезопасным было непростым, но было принято не делать —
эта безопасность не даётся бесплатно. Там где она не нужна, дополнительные средства
синхронизации вроде мьютексов будут излишне замедлять программу, а там где она нужна — не
составляет труда реализовать эту безопасность с помощью sync.Mutex.
